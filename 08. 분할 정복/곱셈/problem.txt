문제 출처 : https://www.acmicpc.net/problem/1629
난이도 : 중

아니 이건 수학 문제자나...

(a * b) % c = ((a % c) * (b % c)) % c
이 공식을 응용하면

# b가 짝수면
(a ** b) = (a**(b//2) * a**(b//2))
# b가 홀수면
(a ** b) = (a**(b//2) * a**(b//2)*a)

(a ** b) % c = ((a**(b//2) % c) * (a**(b//2) % c)) % c
or
(a ** b) % c = ((a**(b//2) % c) * ((a**(b//2)*a) % c)) % c

이것을 코드화 하면 됩답니다...
