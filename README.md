# 백준으로 코딩테스트 완벽가이드 (with python)

백준 단계별 풀어보기의 모든 문제의 해답을 정리해 놓았습니다.
특히 각 단계의 guid에는 template가 준비되어 있습니다. 해당 template는 꼭 외워두시길 바랍니다. 수학의 공식 같은 존재입니다.

링크 : [백준 단계별로 풀어보기](https://www.acmicpc.net/step)
